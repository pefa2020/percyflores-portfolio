## Architecture

This portfolio operates as a blazing-fast, serverless Single-Page Application (SPA) utilizing:
- **Core:** Semantic HTML5, CSS3 Custom Properties, Vanilla ES6 JavaScript
- **Styling:** Custom CSS grid architectures, responsive flexbox fluid layouts, and hardware-accelerated micro-animations (No external libraries/Tailwind)
- **Deployment:** GitHub Pages (CI/CD)

## Repository Structure

- `index.html` — The core application entry point and semantic structure.
- `styles.css` — The custom design system, containing CSS variables, dark-mode styling, and animations.
- `main.js` — The logic engine driving interactive modals, dynamic project filtering, and intersection observer scroll animations.
- `reports/` — Directory containing production-grade HTML versions of academic and security lab reports.
- `assets/` — Core documents, graphics, and resume PDFs.
- `tests/` — Directory containing automated End-to-End (E2E) testing configurations using Playwright.
